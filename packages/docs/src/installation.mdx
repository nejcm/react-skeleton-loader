# Installation

React skeleton supports styled-components and emotion.

## Peer dependencies

React skeleton requires the following peer dependencies to work. Make sure to
have them installed in your project.

<Hint type="info">
  You can <strong>skip this step</strong> if those packages are already
  installed in your application.
</Hint>

```bash
# for styled-components
$ npm install react react-dom styled-components --save

# for emotion
$ npm install react react-dom @emotion/styled @emotion/core --save
```

## Package

Install the latest version of React skeleton by running the following command in
your project's root directory:

```bash
# for styled-components
$ npm install @nejcm/react-skeleton-loader --save

# for emotion
$ npm install @nejcm/react-skeleton-loader-emotion --save
```

## Usage

```jsx
import React from 'react';
// for styled-components
import Skeleton, {Card, Comment} from '@nejcm/react-skeleton-loader';
// for emotion
import Skeleton, {Card, Comment} from '@nejcm/react-skeleton-loader-emotion';

const YourComponent = () => {
  return (
    <Skeleton>
      <Skeleton.Rectangle height={20} />
      <Skeleton.Circle width={40} height={40} />
    </Skeleton>
  );
};

export default YourComponent;
```
