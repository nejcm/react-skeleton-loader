# Skeleton

## Specification

Skeleton is the main component of the library and all skeleton loaders and
templates are built with it.

## Usage

```jsx live
function Example() {
  return (
    <>
      <Skeleton height={30} style={{ marginBottom: 10 }} />
      <Skeleton width={50} height={50} style={{ marginBottom: 10 }} rounded />
      <Skeleton fromColor="#096464" toColor="#075151" width={200} height={45} />
    </>
  );
}
```
